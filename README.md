# HDACs-inhibitors-RNAseq-analysis
RNAseq analysis workflow in R using Bioconductor. The project reanalyzes mouse brain transcriptomic data from Rumbaugh, Gavin et al. “Pharmacological Selectivity Within Class I Histone Deacetylases Predicts Effects on Synaptic Function and Memory Rescue.” as a bioinformatics learning exercise.
